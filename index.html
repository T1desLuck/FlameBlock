FlameBlock Privacy Policy

Last updated: 2026

1. Overview

FlameBlock is an ad and tracker blocking browser extension. It is designed to improve user experience by blocking unwanted content such as advertisements, trackers, pop-ups, and anti-adblock scripts.

2. Data Collection

FlameBlock does not collect, store, or transmit any personal data.

  • No personal information is collected
  • No browsing history is stored externally
  • No analytics or tracking systems are used
  • No data is sent to any server

3. Local Processing

All operations are performed locally within the user's browser:

  • Network requests are filtered using declarativeNetRequest
  • Page elements are modified using local content scripts
  • User settings are stored using browser storage APIs

4. Permissions Usage

declarativeNetRequest

Used to block network requests to known advertising and tracking domains.

scripting

Used to inject scripts for hiding ads and preventing pop-ups.

storage

Used to store user preferences locally (e.g., whitelist, settings).

tabs

Used to determine the active tab and apply site-specific settings.

webNavigation

Used to detect page navigation and apply filtering rules.

webRequest

Used only for observing blocked requests and counting them for display purposes.

5. Host Permissions

FlameBlock requires access to all websites (<all_urls>) in order to block ads and trackers across the web.

6. Remote Code

FlameBlock does not use remote code.

  • No scripts are downloaded from external servers
  • All logic is bundled inside the extension

7. Third-Party Services

FlameBlock does not use any third-party services, analytics platforms, or external APIs.

8. Security

All processing is performed locally, reducing exposure of user data.

9. Changes to This Policy

This privacy policy may be updated when the extension is updated. Changes will be reflected on this page.

10. Contact

If you have any questions, you can contact the developer via the extension listing page.
